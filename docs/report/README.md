# 逸仙活动云课程报告

本目录包含软件工程期末大作业所需的六份 LaTeX 报告。每份报告均在独立文件夹中，入口均为 `main.tex`；公共排版文件为 `report-common.tex`。

使用 XeLaTeX 编译：在任一报告目录执行 `xelatex -interaction=nonstopmode main.tex` 两次。

提交前请替换封面中的组长学号、姓名、教师和日期；PDF 命名时按课程要求采用“组长学号组长姓名报告名称.pdf”。

| 文件夹 | 报告 |
|---|---|
| `01-requirements` | 需求分析报告 |
| `02-modeling` | 系统建模报告 |
| `03-architecture` | 架构设计文档 |
| `04-engineering` | 软件工程化说明文档 |
| `05-testing-quality` | 软件测试与质量保证报告 |
| `06-configuration-operations` | 软件配置与运维文档 |
