# 搜索页面设计

## 目标与入口

- 路径：`/search`；访客可用。
- 入口：首页顶栏搜索框和公开活动列表筛选。
- 通过 URL 保存 `q`、`scope`（internal/external）、`page`、`activity_type` 和 `sort`，链接可直接复现结果。

## 布局与状态

- 沿用深绿顶栏、浅绿背景和白色圆角卡片；主体为筛选侧栏和结果区，小屏收为纵向。
- 结果卡展示标题、摘要、类型、时间、地点和主办方；外部结果保留来源标记和安全外链。
- 明确支持空关键词、加载、无结果、服务失败和外部搜索不可用状态。

## 接口与验收

- `GET /search/internal`、`GET /search/external`，参数为 `q`、`page`、`per_page`、`activity_type`、`sort`。
- 搜索输入防抖只作用于建议/自动查询，提交和筛选更新 URL。
- 验收：刷新或分享 URL 后恢复筛选；内部和外部请求错误互不影响。
